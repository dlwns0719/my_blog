лӘ©м°Ё

  • 1 нҷҳкІҪ м„Өм •
  • 2 кё°ліё мӮ¬мҡ©лІ•: л‘җ к°ңмқҳ кІҖмғүкё° кІ°н•©
    • 2.1 мҳҲм ң 1: нӮӨмӣҢл“ң кё°л°ҳ кІҖмғүм—җ мң лҰ¬н•ң мҝјлҰ¬
    • 2.2 мҳҲм ң 2: мқҳлҜё кІҖмғүмқҙ лҚ” мң мҡ©н•ң ліөмһЎн•ң мҝјлҰ¬
  • 3 кі кёү кё°лҠҘ: лҹ°нғҖмһ„м—җ к°ҖмӨ‘м№ҳ лҸҷм Ғ ліҖкІҪ
    • 3.1 лҸҷм Ғ к°ҖмӨ‘м№ҳ м„Өм • кө¬м„ұ
    • 3.2 мјҖмқҙмҠӨ 2: FAISS к°ҖмӨ‘м№ҳ к·№лҢҖнҷ” (мқҳлҜё кё°л°ҳ кІҖмғү)

EnsembleRetrieverлҠ” м—¬лҹ¬ кІҖмғүкё°лҘј кІ°н•©н•ҳм—¬ лҚ” к°•л Ҙн•ң кІҖмғү кІ°кіјлҘј м ңкіөн•ҳлҠ” LangChainмқҳ кё°лҠҘмқҙлӢӨ. лӢӨм–‘н•ң кІҖмғү м•Ңкі лҰ¬мҰҳмқҳ мһҘм җмқ„ нҷңмҡ©н•ҳм—¬ лӢЁмқј м•Ңкі лҰ¬мҰҳліҙлӢӨ лҚ” лӮҳмқҖ м„ұлҠҘмқ„ лӢ¬м„ұн•  мҲҳ мһҲмңјл©°, мӮ¬мҡ©мһҗл“Ө мӮ¬мқҙм—җм„ң лҶ’мқҖ нҳёнҸүмқ„ л°ӣкі  мһҲлӢӨ. лӢӨмҲҳмқҳ лҰ¬нҠёлҰ¬лІ„лҘј мӮ¬мҡ©н•ҳм—¬ лӮҳмҳЁ кІ°кіјлҘј Reciprocal Rank Fusion(RRF) м•Ңкі лҰ¬мҰҳмңјлЎң мһ¬мҲңмң„нҷ”н•ң нӣ„ н•ҳлӮҳмқҳ нҶөн•© кІҖмғү кІ°кіјлҘј LLMмқҳ мһ…л ҘмңјлЎң м ңкіөн•ңлӢӨ.

мЈјмҡ” нҠ№м§•

  1. м—¬лҹ¬ кІҖмғүкё° нҶөн•©: лӢӨм–‘н•ң мң нҳ•мқҳ кІҖмғүкё°лҘј мһ…л ҘмңјлЎң л°ӣм•„ кІ°кіјлҘј кІ°н•©н•ңлӢӨ.

  2. кІ°кіј мһ¬мҲңмң„нҷ” (Reciprocal Rank Fusion): RRF м•Ңкі лҰ¬мҰҳмқ„ мӮ¬мҡ©н•ҳм—¬ к°Ғ кІҖмғүкё°мқҳ кІ°кіјлҘј лӢЁмҲң н•©м§‘н•©мқҙ м•„лӢҢ м§ҖлҠҘнҳ• мҲңмң„ л§Өк№Җмқ„ нҶөн•ҙ нҶөн•©н•ңлӢӨ.

    • к°Ғ кІҖмғүкё°л“Өмқҙ кІҖмғүн•ң мң мӮ¬лҸ„ лҶ’мқҖ л¬ём„ңл“Өмқ„ лӢЁмҲңнһҲ н•©м№ҳм§Җ м•Ҡкі  лӢӨмӢң мҲңмң„лҘј л§ӨкёҙлӢӨ
    • Agent к°ңл°ңмһҗл“Ө мӮ¬мқҙм—җм„ңлҠ” EnsembleRetriever(LangChain мҡ©м–ҙ)ліҙлӢӨ RRF(мқјл°ҳ нҶөмҡ© мҡ©м–ҙ)лҘј нӣЁм”¬ лҚ” мқјл°ҳм ҒмңјлЎң мӮ¬мҡ©н•ңлӢӨ
    • л…јл¬ём—җм„ң кұ°мқҳ лӘЁл“  кІҪмҡ°м—җ RRFк°Җ лҚ” мўӢмқҖ м„ұлҠҘмқ„ ліҙмһ„мқ„ ліҙм—¬мӨҖлӢӨ
    • мӢӨм ң м Ғмҡ©м—җм„ң RRFмқҳ нҡЁкіјлҠ” л№„мҠ·н•ң нҠ№м„ұмқҳ лҰ¬нҠёлҰ¬лІ„лҘј кІ°н•©н•  л•ҢліҙлӢӨ лӢӨм–‘н•ң нҠ№м„ұмқҳ лҰ¬нҠёлҰ¬лІ„лҘј кІ°н•©н•  л•Ң к·№лҢҖнҷ”лҗңлӢӨ

  3. н•ҳмқҙлёҢлҰ¬л“ң кІҖмғү: мЈјлЎң sparse retriever(мҳҲ: BM25)мҷҖ dense retriever(мҳҲ: мһ„лІ л”© мң мӮ¬лҸ„)лҘј кІ°н•©н•ҳм—¬ мӮ¬мҡ©н•ңлӢӨ.

к°Ғ кІҖмғүкё°мқҳ мһҘм җ

  • Sparse Retriever (BM25): нӮӨмӣҢл“ң кё°л°ҳ кІҖмғүм—җ нҡЁкіјм Ғ вҶ’ мӮ¬мҡ©мһҗмқҳ м •нҷ•н•ң мҡ©м–ҙм—җ лҜјк°җн•ҳкІҢ л°ҳмқ‘
  • Dense Retriever (мһ„лІ л”© кё°л°ҳ): мқҳлҜём Ғ мң мӮ¬м„ұ кё°л°ҳ кІҖмғүм—җ нҡЁкіјм Ғ вҶ’ мқҳлҸ„мҷҖ л§ҘлқҪмқ„ мқҙн•ҙн•ң кІҖмғү

мқҙлҹ¬н•ң мғҒнҳё ліҙмҷ„м Ғмқё нҠ№м„ұмңјлЎң мқён•ҙ EnsembleRetrieverлҠ” лӢӨм–‘н•ң кІҖмғү мӢңлӮҳлҰ¬мҳӨм—җм„ң н–ҘмғҒлҗң м„ұлҠҘмқ„ м ңкіөн•ңлӢӨ.

мһҗм„ён•ң лӮҙмҡ©мқҖ LangChain кіөмӢқ л¬ём„ңлҘј м°ёмЎ°н•ҳмһҗ.

1 нҷҳкІҪ м„Өм •

мҪ”л“ң 
# API нӮӨлҘј нҷҳкІҪліҖмҲҳлЎң кҙҖлҰ¬н•ҳкё° мң„н•ң м„Өм •
from dotenv import load_dotenv

# API нӮӨ м •ліҙ лЎңл“ң
load_dotenv()
мҪ”л“ң 
# LangSmith м¶”м Ғмқ„ м„Өм •н•©лӢҲлӢӨ. https://smith.langchain.com
# !pip install langchain-teddynote
from langchain_teddynote import logging

# н”„лЎңм қнҠё мқҙлҰ„мқ„ мһ…л Ҙ
logging.langsmith("CH10-Retriever")

2 кё°ліё мӮ¬мҡ©лІ•: л‘җ к°ңмқҳ кІҖмғүкё° кІ°н•©

EnsembleRetrieverлҘј мҙҲкё°нҷ”н•ҳм—¬ BM25RetrieverмҷҖ FAISS кІҖмғүкё°лҘј кІ°н•©н•ңлӢӨ. к°Ғ кІҖмғүкё°лҠ” кі мң н•ң кІҖмғү л°©мӢқмқ„ мӮ¬мҡ©н•ңлӢӨ: - BM25Retriever: нӮӨмӣҢл“ң кё°л°ҳ кІҖмғү (sparse search) - FAISS: мһ„лІ л”© кё°л°ҳ мқҳлҜё кІҖмғү (dense search)

мҪ”л“ң 
from langchain.retrievers import BM25Retriever, EnsembleRetriever
from langchain.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings

# мғҳн”Ң л¬ём„ң лҚ°мқҙн„°м…Ӣ (Apple кҙҖл Ё л¬ём„ңл“Ө)
doc_list = [
    "I like apples",
    "I like apple company",
    "I like apple's iphone",
    "Apple is my favorite company",
    "I like apple's ipad",
    "I like apple's macbook",
]

# BM25 Retriever мҙҲкё°нҷ” (нӮӨмӣҢл“ң кё°л°ҳ кІҖмғү)
bm25_retriever = BM25Retriever.from_texts(doc_list)
bm25_retriever.k = 1  # кІҖмғү кІ°кіј к°ңмҲҳ

# FAISS Retriever мҙҲкё°нҷ” (мқҳлҜё кё°л°ҳ кІҖмғү)
embedding = OpenAIEmbeddings()
faiss_vectorstore = FAISS.from_texts(doc_list, embedding)
faiss_retriever = faiss_vectorstore.as_retriever(search_kwargs={"k": 1})

# Ensemble Retriever мғқм„ұ (к°ҖмӨ‘м№ҳ: BM25 70%, FAISS 30%)
ensemble_retriever = EnsembleRetriever(
    retrievers=[bm25_retriever, faiss_retriever],
    weights=[0.7, 0.3],  # BM25м—җ лҚ” лҶ’мқҖ к°ҖмӨ‘м№ҳ л¶Җм—¬
)

2.1 мҳҲм ң 1: нӮӨмӣҢл“ң кё°л°ҳ кІҖмғүм—җ мң лҰ¬н•ң мҝјлҰ¬

ensemble_retriever к°қмІҙлҘј мӮ¬мҡ©н•ҳм—¬ кҙҖл Ё л¬ём„ңлҘј кІҖмғүн•ңлӢӨ. к°Ғ кІҖмғүкё°мқҳ кІ°кіјлҘј л№„көҗн•ҳл©ҙ к°ҖмӨ‘м№ҳмқҳ мҳҒн–Ҙмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

мҪ”л“ң 
# кІҖмғү мӢӨн–ү
query = "my favorite fruit is apple"
ensemble_result = ensemble_retriever.invoke(query)
bm25_result = bm25_retriever.invoke(query)
faiss_result = faiss_retriever.invoke(query)

# кІ°кіј м¶ңл Ҙ
print("[Ensemble Retriever кІ°кіј]")
for doc in ensemble_result:
    print(f"Content: {doc.page_content}")
    print()

print("[BM25 Retriever кІ°кіј]")
for doc in bm25_result:
    print(f"Content: {doc.page_content}")
    print()

print("[FAISS Retriever кІ°кіј]")
for doc in faiss_result:
    print(f"Content: {doc.page_content}")
    print()

кІ°кіј н•ҙм„қ: - BM25лҠ” м •нҷ•н•ң нӮӨмӣҢл“ң л§Өм№ӯ(вҖңappleвҖқкіј вҖңfruitвҖқ)мңјлЎң вҖңI like applesвҖқ л¬ём„ңлҘј м°ҫмқҢ - FAISSлҠ” мқҳлҜё мң мӮ¬м„ұмңјлЎң вҖңfavoriteвҖқкіј кҙҖл Ёлҗң вҖңApple is my favorite companyвҖқ л¬ём„ңлҘј м°ҫмқҢ - EnsembleмқҖ BM25 к°ҖмӨ‘м№ҳ(0.7)к°Җ лҶ’м•„м„ң BM25 кІ°кіјлҘј мҡ°м„ н•Ё

2.2 мҳҲм ң 2: мқҳлҜё кІҖмғүмқҙ лҚ” мң мҡ©н•ң ліөмһЎн•ң мҝјлҰ¬

лҚ” мһҗм—°мҠӨлҹ¬мҡҙ л¬ёмһҘ нҳ•нғңмқҳ мҝјлҰ¬лЎң л‘җ кІҖмғү л°©мӢқмқҳ м°ЁмқҙлҘј кҙҖм°°н•ңлӢӨ.

мҪ”л“ң 
# ліөмһЎн•ң мҝјлҰ¬ мӢӨн–ү
query = "Apple company makes my favorite iphone"
ensemble_result = ensemble_retriever.invoke(query)
bm25_result = bm25_retriever.invoke(query)
faiss_result = faiss_retriever.invoke(query)

# кІ°кіј м¶ңл Ҙ
print("[Ensemble Retriever кІ°кіј]")
for doc in ensemble_result:
    print(f"Content: {doc.page_content}")
    print()

print("[BM25 Retriever]")
for doc in bm25_result:
    print(f"Content: {doc.page_content}")
    print()

print("[FAISS Retriever кІ°кіј]")
for doc in faiss_result:
    print(f"Content: {doc.page_content}")
    print()

кІ°кіј н•ҙм„қ: - BM25лҠ” вҖңAppleвҖқкіј вҖңcompanyвҖқлқјлҠ” м •нҷ•н•ң нӮӨмӣҢл“ңм—җл§Ң л°ҳмқ‘ вҶ’ н•ң л¬ём„ңл§Ң л°ҳнҷҳ - FAISSлҠ” вҖңfavoriteвҖқкіј вҖңiphoneвҖқмқҳ мқҳлҜё кҙҖкі„лҘј мқҙн•ҙ вҶ’ лҚ” м •нҷ•н•ң л¬ём„ң кІҖмғү - EnsembleмқҖ л‘җ кІ°кіјлҘј кІ°н•© вҶ’ лӢӨм–‘н•ң кҙҖм җм—җм„ңмқҳ м •ліҙ м ңкіө - мӢӨм ң мӮ¬мҡ©м—җм„ңлҠ” мқҳлҜё кІҖмғүмқҳ к°ҖмӨ‘м№ҳлҘј лҶ’мқҙлҠ” кІғмқҙ ліөмһЎн•ң мҝјлҰ¬м—җм„ң лҚ” нҡЁкіјм Ғ

3 кі кёү кё°лҠҘ: лҹ°нғҖмһ„м—җ к°ҖмӨ‘м№ҳ лҸҷм Ғ ліҖкІҪ

ConfigurableFieldлҘј мӮ¬мҡ©н•ҳл©ҙ н”„лЎңк·ёлһЁ мӢӨн–ү мӨ‘(лҹ°нғҖмһ„)м—җ к°Ғ кІҖмғүкё°мқҳ к°ҖмӨ‘м№ҳлҘј лҸҷм ҒмңјлЎң ліҖкІҪн•  мҲҳ мһҲлӢӨ. мқҙлҠ” лӢӨм–‘н•ң мҝјлҰ¬ мң нҳ•м—җ лҢҖмқ‘н•ҳлҠ” мң м—°н•ң кІҖмғү мӢңмҠӨн…ңмқ„ кө¬м¶•н•  мҲҳ мһҲлӢӨлҠ” мһҘм җмқҙ мһҲлӢӨ.

3.1 лҸҷм Ғ к°ҖмӨ‘м№ҳ м„Өм • кө¬м„ұ

мҪ”л“ң 
from langchain_core.runnables import ConfigurableField

# ConfigurableFieldлҘј мӮ¬мҡ©н•ң лҸҷм Ғ к°ҖмӨ‘м№ҳ м„Өм •
ensemble_retriever = EnsembleRetriever(
    # лҰ¬нҠёлҰ¬лІ„ лӘ©лЎқмқ„ м„Өм •н•©лӢҲлӢӨ. м—¬кё°м„ңлҠ” bm25_retrieverмҷҖ faiss_retrieverлҘј мӮ¬мҡ©н•©лӢҲлӢӨ.
    retrievers=[bm25_retriever, faiss_retriever],
).configurable_fields(
    weights=ConfigurableField(
        id="ensemble_weights",  # мҷёл¶Җм—җм„ң м°ёмЎ°н•  мӢқлі„мһҗ
        name="Ensemble Weights",
        description="BM25мҷҖ FAISS кІҖмғүкё°мқҳ к°ҖмӨ‘м№ҳ л№„мңЁ (н•©кі„=1.0)",
    )
)
  • кІҖмғү мӢң config л§Өк°ңліҖмҲҳлҘј нҶөн•ҙ кІҖмғү м„Өм •мқ„ м§Җм •
    • ensemble_weights мҳөм…ҳмқҳ к°ҖмӨ‘м№ҳлҘј [1, 0]мңјлЎң м„Өм •н•ҳм—¬ лӘЁл“  кІҖмғү кІ°кіјмқҳ к°ҖмӨ‘м№ҳк°Җ BM25 retriever м—җ лҚ” л§Һмқҙ л¶Җм—¬ лҗҳлҸ„лЎқ
мҪ”л“ң 
# BM25 м „мҡ© кІҖмғү м„Өм •
config = {"configurable": {"ensemble_weights": [1, 0]}}

# лҸҷмқјн•ң мҝјлҰ¬лЎң кІҖмғү мӢӨн–ү
docs = ensemble_retriever.invoke("my favorite fruit is apple", config=config)

print("[BM25л§Ң мӮ¬мҡ© (к°ҖмӨ‘м№ҳ [1.0, 0.0])]")
for i, doc in enumerate(docs, 1):
    print(f"{i}. {doc.page_content}")
[BM25л§Ң мӮ¬мҡ© (к°ҖмӨ‘м№ҳ [1.0, 0.0])]
1. I like apples

кІ°кіј н•ҙм„қ: BM25л§Ң мӮ¬мҡ©н•  л•ҢлҠ” м •нҷ•н•ң нӮӨмӣҢл“ң л§Өм№ӯл§Ң мқҙлЈЁм–ҙм§җ. вҖңappleвҖқкіј вҖңfruitвҖқмқ„ нҸ¬н•Ён•ң л¬ём„ңл§Ң л°ҳнҷҳлҗЁ.

3.2 мјҖмқҙмҠӨ 2: FAISS к°ҖмӨ‘м№ҳ к·№лҢҖнҷ” (мқҳлҜё кё°л°ҳ кІҖмғү)

FAISS кІҖмғүкё°м—җ м „мІҙ к°ҖмӨ‘м№ҳ(1.0)лҘј н• лӢ№н•ҳм—¬ мқҳлҜё кё°л°ҳ кІҖмғүл§Ң мҲҳн–үн•ңлӢӨ.

мҪ”л“ң 
# FAISS м „мҡ© кІҖмғү м„Өм •
config = {"configurable": {"ensemble_weights": [0, 1]}}

# лҸҷмқјн•ң мҝјлҰ¬лЎң кІҖмғү мӢӨн–ү
docs = ensemble_retriever.invoke("my favorite fruit is apple", config=config)

print("[FAISSл§Ң мӮ¬мҡ© (к°ҖмӨ‘м№ҳ [0.0, 1.0])]")
for i, doc in enumerate(docs, 1):
    print(f"{i}. {doc.page_content}")
[FAISSл§Ң мӮ¬мҡ© (к°ҖмӨ‘м№ҳ [0.0, 1.0])]
1. Apple is my favorite company

кІ°кіј н•ҙм„қ: FAISSл§Ң мӮ¬мҡ©н•  л•ҢлҠ” мқҳлҜё мң мӮ¬м„ұмңјлЎң кІҖмғү. вҖңmy favoriteвҖқмқҙлқјлҠ” н‘ңнҳ„мқ„ вҖңApple is my favorite companyвҖқм—җм„ң м°ҫм•„ л°ҳнҷҳн•Ё.
